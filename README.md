# Password Cracking Analysis

This repository contains my assignment on *Password Cracking Analysis* for practical lab and theory work.

## Description
This project covers:
- Comparison of different hashing algorithms (MD5, SHA1, SHA256, bcrypt, Argon2)
- Practical cracking of weak hashes using *John the Ripper*
- Explanation of why some hashes are more secure than others

## Tools Used
- Kali Linux
- John the Ripper
- Terminal / Command-line
